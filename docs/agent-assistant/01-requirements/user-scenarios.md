## 二、HOW_TO — 平台功能操作指引

### S-H1: 怎么成为创作者
- **用户问**：「怎么成为创作者？」
- **意图**：HOW_TO
- **grounding**：帮助中心知识库「创作者认证」条目
- **期望回答**：说明条件（总获赞≥10000 且发帖≥50）+ 申请入口（设置→创作者认证 或 /creator-verification）+ 申请材料（真实姓名+身份证号）
- **跳转**：`/creator-verification`

### S-H2: 怎么关闭通知
- **用户问**：「怎么关掉点赞通知的红点？」
- **意图**：HOW_TO
- **grounding**：帮助中心「通知偏好设置」条目
- **期望回答**：路径「我的→设置→通知偏好→关闭点赞收藏通知」+ 说明关闭后不推红点但收纳篮仍可见
- **跳转**：`/settings/notifications`

### S-H3: 怎么隐藏私信
- **用户问**：「怎么把一条私信隐藏掉而不是删除？」
- **grounding**：帮助中心「私信隐藏」条目
- **跳转**：`/messages`

### S-H4: 怎么改头像
- **用户问**：「头像怎么换成自己的照片？」
- **grounding**：帮助中心「个人资料」条目
- **跳转**：`/profile`（编辑资料）

### S-H5: 怎么发资源贴
- **用户问**：「我想分享一份 PDF 资料，怎么发？」
- **grounding**：帮助中心「发帖」条目
- **期望**：说明选择学校/分类→发帖→选「资源贴」→上传文件（≤100MB）

### S-H6: 怎么看浏览历史
- S-H7: 怎么关注别人
- S-H8: 怎么修改密码
- S-H9: 金色 V 是什么（创作者认证解释）
- S-H10: 私信为什么只能发一条（单向消息限制解释）
- S-H11: 怎么设置不让别人看到我的收藏（隐私开关）
- S-H12: 忘记密码怎么办

> 每条均映射到 `05-knowledge-base/knowledge-sources.md` 的帮助文档条目。

## 三、SEARCH — 资源/讨论帖检索

### S-R1: 跨校资源检索
- **用户问**：「求清华操作系统期末复习资料」
- **意图**：SEARCH
- **grounding**：post-service 帖子检索（学校=清华 + 语义=操作系统期末复习）
- **期望**：返回资源贴卡片列表（标题/作者/下载信息）+ 引用，点击跳 `/school/:id/post/:id`
- **若召回为空**：说明未找到，推荐相近板块（如「面经」分类的 OS 面试题）

### S-R2: 跨分类语义检索
- **用户问**：「有没有 K-POP 的安利帖」
- **grounding**：分类=音乐→K-POP 子板块 + 语义检索
- **跳转**：`/category/:id`

### S-R3: 讨论贴检索
- **用户问**：「想讨论 2026 秋招面经」
- **grounding**：分类=面经 + 讨论贴类型过滤 + 语义

### S-R4: 同义词/缩写理解
- **用户问**：「有没有 OS 的笔记」（OS=操作系统）
- **期望**：查询扩展后命中「操作系统」相关帖

### S-R5: 模糊需求澄清
- **用户问**：「求点复习资料」（没说科目）
- **期望**：Agent 反问澄清「请问是哪个学校、哪门课的复习资料？」

### S-R6: 多条件组合
- **用户问**：「北大 计算机网络 期末 卷子」
- **期望**：组合过滤（学校=北大 + 语义=计算机网络期末卷）

### S-R7: 排序偏好
- **用户问**：「最热的操作系统讨论」
- **期望**：按 like_count/comment_count 排序

### S-R8: 帖子内容问答
- **用户问**：「那个讲 TCP 三次握手的帖子里说了什么？」
- **期望**：先检索定位帖子，再抽取内容摘要回答 + 引用

## 五、CLARIFY — 多轮澄清与追问

### S-C1: 指代消解
- T1:「求 OS 资料」→ 返回清华帖
- T2:「那个有下载链接的是哪个」→ 指代上一轮结果，定位带文件的帖

### S-C2: 追问深入
- T1:「怎么成为创作者」
- T2:「我现在差多少」→ Agent 调用工具查用户当前获赞/发帖数，给出差距

### S-C3: 修正
- T1:「求北大操作系统资料」
- T2:「我说的是清华」→ 修正后重检索

### S-C4: 换主题
- T1:「求 OS 资料」
- T2:「顺便问下怎么关通知」→ 上下文切换，识别新意图

## 六、OUT_OF_SCOPE — 超范围拒绝

### S-O1: 闲聊
- 「你是谁」「今天天气怎么样」→ 礼貌说明只服务 CampusShare 平台问题

### S-O2: 开放域知识
- 「量子力学是什么」→ 拒绝，引导回平台

### S-O3: 写操作请求
- 「帮我发个帖子」「帮我点赞」→ 拒绝代操作，给指引

### S-O4: 敏感
- 政治/医疗/法律建议 → 拒绝
