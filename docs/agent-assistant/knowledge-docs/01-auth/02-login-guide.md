# 登录指南

## 登录方式

CampusShare 支持**三种**登录方式，使用同一个账号输入框：

### 1. 用户名登录
- 输入注册时填写的用户名 + 密码

### 2. 手机号登录
- 输入注册时使用的手机号 + 密码

### 3. 邮箱登录
- 输入注册时使用的邮箱 + 密码

## 登录流程

1. 在登录页面，账号输入框中输入用户名、手机号或邮箱
2. 输入密码
3. 可选：勾选"记住我"保持登录状态
4. 点击"登录"按钮

## 记住我功能

- 勾选"记住我"后，登录状态会在较长时间内保持
- 未勾选时，关闭浏览器后可能需要重新登录

## 注意事项

- 系统会自动识别输入的是用户名、手机号还是邮箱
- 忘记密码请联系管理员重置
- 登录成功后会获取 JWT token，用于后续接口鉴权
