# 注册指南

## 注册方式

CampusShare 支持两种注册方式：

### 1. 手机号注册
需要提供以下信息：
- **手机号**：作为登录账号使用
- **用户名**：展示名称，用于在平台上显示
- **密码**：登录凭证
- **短信验证码**：发送到手机号，用于验证真实性

### 2. 邮箱注册
需要提供以下信息：
- **邮箱**：作为登录账号使用
- **用户名**：展示名称，用于在平台上显示
- **密码**：登录凭证
- **邮箱验证码**：发送到邮箱，用于验证真实性

## 注册流程

1. 在登录页面点击"注册"按钮
2. 选择注册方式（手机号或邮箱）
3. 填写账号（手机号/邮箱）、用户名、密码
4. 点击"发送验证码"，等待接收
5. 输入收到的验证码
6. 点击"注册"完成

## 注意事项

- 用户名是必填项，注册后可在个人中心修改
- 手机号和邮箱均可用于登录，但注册时只能选择一种
- 验证码有有效期，请及时输入
- 密码需要满足安全要求
