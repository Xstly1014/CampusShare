# CampusShare 平台介绍

## 平台定位

CampusShare（校园资源共享平台）是一个面向校园用户的资源共享与交流平台，旨在帮助校园用户：
- 分享学习资料和资源
- 交流学习经验和生活话题
- 按学校和分类浏览感兴趣的内容

## 核心功能

### 用户系统
- 支持**手机号**和**邮箱**两种注册方式
- 支持**手机号、邮箱**两种登录方式
- 用户角色：普通用户、创作者、管理员
- 创作者等级：初级、中级、高级、权威

### 帖子系统
- 两种帖子类型：资源（resource）和讨论（discussion）
- 按学校和分类体系组织内容
- 支持文件上传（最大 100MB）
- 多种排序方式：最新、最热、最活跃

### 互动系统
- 点赞与收藏
- 评论与回复（支持多级回复）
- 浏览历史记录
- 下载记录

### 通知系统
- 多种通知类型（点赞、评论、收藏、关注等）
- 通知聚合与独立展示
- 已读/未读管理
- 通知开关设置

### 智能助手
- AI 智能问答助手"小享"
- 基于平台真实功能回答问题
- RAG 检索增强，支持知识库和帖子搜索

## 技术架构

- 前端：React + TypeScript + Vite
- 后端：Spring Cloud 微服务架构
  - gateway-service：API 网关 + JWT 认证
  - user-service：用户认证与管理
  - post-service：帖子与评论
  - agent-service：智能助手
- 数据库：MySQL（业务数据）+ PostgreSQL+pgvector（向量数据）+ Redis（缓存）
- 部署：Docker Compose
