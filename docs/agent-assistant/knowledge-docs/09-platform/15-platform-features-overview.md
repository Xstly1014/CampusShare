# CampusShare 功能总览

## 认证功能

| 功能 | 说明 |
|------|------|
| 手机号注册 | 使用手机号 + 验证码注册 |
| 邮箱注册 | 使用邮箱 + 验证码注册 |
| 手机号登录 | 使用手机号 + 密码登录 |
| 邮箱登录 | 使用邮箱 + 密码登录 |
| 记住我 | 保持登录状态 |
| 修改密码 | 在个人中心修改密码 |
| 实名认证 | 实名验证获取更多权限 |

## 帖子功能

| 功能 | 说明 |
|------|------|
| 发布资源帖 | 分享学习资料、文件 |
| 发布讨论帖 | 发起话题讨论 |
| 编辑帖子 | 修改自己的帖子 |
| 删除帖子 | 删除自己的帖子 |
| 按学校浏览 | 筛选特定学校的帖子 |
| 按分类浏览 | 按主分类/子分类筛选 |
| 关键词搜索 | 在分类内搜索帖子 |
| 多种排序 | 最新/最热/最活跃 |

## 文件功能

| 功能 | 说明 |
|------|------|
| 文件上传 | 最大 100MB，multipart 格式 |
| 文件下载 | 资源帖附带文件下载 |
| 下载记录 | 查看历史下载 |

## 互动功能

| 功能 | 说明 |
|------|------|
| 点赞帖子 | 切换点赞状态 |
| 收藏帖子 | 切换收藏状态 |
| 发表评论 | 对帖子发表评论 |
| 回复评论 | 回复其他用户的评论 |
| 点赞评论 | 对评论点赞 |
| 删除评论 | 删除自己的评论 |

## 个人中心

| 功能 | 说明 |
|------|------|
| 个人主页 | 展示用户信息和统计 |
| 编辑资料 | 修改头像、昵称、简介 |
| 我的发布 | 查看自己发布的帖子 |
| 我的收藏 | 查看收藏的帖子 |
| 我的点赞 | 查看点赞的帖子 |
| 浏览历史 | 查看浏览记录 |
| 我的评论 | 查看发表的评论 |
| 隐私设置 | 控制信息可见性 |
| 通知设置 | 配置通知开关 |

## 通知功能

| 功能 | 说明 |
|------|------|
| 通知列表 | 查看所有通知 |
| 按类型筛选 | 按通知类型查看 |
| 标记已读 | 单条或批量标记 |
| 未读计数 | 底部导航栏显示 |
| 通知置顶 | 置顶重要通知 |

## 智能助手

| 功能 | 说明 |
|------|------|
| AI 问答 | 智能助手小享回答问题 |
| 流式输出 | 实时流式返回回答 |
| 会话管理 | 新建/切换/归档/删除会话 |
| 知识库检索 | 基于平台真实功能回答 |
| 帖子检索 | 搜索平台帖子内容 |
