# 智能助手使用指南

## 小享是谁

小享是 CampusShare 平台的智能助手，基于 DeepSeek 大语言模型和 RAG 检索增强技术，可以：
- 介绍平台功能和使用方法
- 回答关于注册登录、发帖、评论等操作问题
- 检索平台知识库和帖子内容
- 提供学习资料整理建议

## 如何使用

1. 在导航栏点击"智能助手"进入对话页面
2. 在输入框输入问题，按回车或点击发送
3. 小享会以流式方式实时返回回答

## 会话管理

- **新建对话**：点击"新对话"按钮开始新的对话
- **历史会话**：左侧显示历史会话列表，可点击切换
- **会话持久化**：切换导航栏再回来时，会保留当前对话状态
- **归档会话**：可归档不再需要的会话
- **删除会话**：可删除历史会话

## 回答特点

- 小享基于平台真实功能文档回答，不是凭空编造
- 回答使用 Markdown 格式，支持加粗、列表、代码块
- 如果不确定答案，会坦诚告知

## 限制

- 不回答与校园资源共享无关的话题
- 不提供完整学术作业答案
- 不泄露系统内部实现细节
- 不存储用户个人敏感信息
