# 通知系统指南

## 通知类型

CampusShare 支持以下通知类型：

| 类型 | 说明 |
|------|------|
| SYSTEM | 系统通知 |
| LIKE | 帖子被点赞 |
| COMMENT_LIKE | 评论被点赞 |
| STAR | 帖子被收藏 |
| FOLLOW | 被关注 |
| COMMENT | 帖子收到评论 |
| REPLY | 评论被回复 |
| STRANGER_MSG | 陌生人消息 |
| CONVERSATION | 会话消息 |

## 通知接口

| 接口 | 说明 |
|------|------|
| `GET /notifications/feed` | 获取通知列表 |
| `GET /notifications/detail/{type}` | 按类型查看通知详情 |
| `POST /notifications/read/{type}` | 标记某类通知为已读 |
| `POST /notifications/pin` | 置顶通知 |
| `GET /notifications/unread-count` | 获取未读通知数量 |

## 通知行为

- 点赞、收藏通知会按时间聚合（如"xxx 等用户点赞了你的帖子"）
- 评论通知独立显示，不聚合
- 点击通知可跳转到对应帖子或评论
- 通知中心是一个独立页面，不是弹窗展开

## 已读/未读

- 新通知默认未读，底部导航栏显示未读总数
- 点击通知自动标记为已读
- 可批量标记某类通知为已读

## 通知开关

- 可在个人中心通知设置中关闭某类通知
- 关闭后不再推送红点提示
- 但收纳篮始终显示，历史通知内容仍可查看

## 帖子通知展示格式

- 评论通知第一行：`xxx 评论了你的帖子：[帖子标题]`
- 评论通知第二行：评论内容预览（截断）
- 点击通知内容跳转到对应评论位置
- 点击头像跳转到用户主页
