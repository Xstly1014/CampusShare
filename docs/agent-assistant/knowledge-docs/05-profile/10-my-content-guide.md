# 我的内容指南

## 我的发布

- 查看自己发布的所有帖子
- 支持按帖子类型筛选
- 支持分页

## 我的收藏

- 接口：`GET /posts/starred`
- 查看自己收藏的所有帖子
- 支持分页

## 我的点赞

- 接口：`GET /posts/liked`
- 查看自己点赞过的所有帖子
- 支持分页

## 浏览历史

- 查看自己的帖子浏览记录
- 系统自动记录浏览过的帖子
- 支持分页

## 我的评论

- 接口：`GET /posts/my-comments`
- 查看自己发表的所有评论
- 包含评论所属的帖子信息

## 我的下载

- 接口：`GET /posts/my-downloads`
- 查看自己下载过的文件记录

## 帖子统计

- 接口：`GET /users/{userId}/stats`（通过内部API）
- 包含发帖数、总浏览量、总点赞数、总收藏数等统计信息
