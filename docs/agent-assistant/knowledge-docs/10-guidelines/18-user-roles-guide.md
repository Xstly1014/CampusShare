# 用户角色与权限指南

## 用户角色

CampusShare 有三种用户角色：

| 角色 | 标识 | 说明 |
|------|------|------|
| 普通用户 | USER | 注册后的默认角色，拥有基本功能权限 |
| 创作者 | CREATOR | 内容创作者，拥有更高等级权限 |
| 管理员 | ADMIN | 平台管理员，拥有管理权限 |

## 创作者等级

创作者分为以下等级：

| 等级 | 标识 | 说明 |
|------|------|------|
| 无 | NONE | 非创作者 |
| 初级 | JUNIOR | 初级创作者 |
| 中级 | INTERMEDIATE | 中级创作者 |
| 高级 | SENIOR | 高级创作者 |
| 权威 | AUTHORITY | 权威创作者 |

## 各角色权限

### 普通用户（USER）
- 发布帖子（资源/讨论）
- 上传文件（100MB 限制）
- 评论与回复
- 点赞与收藏
- 查看个人主页
- 使用智能助手
- 管理自己的内容和通知

### 创作者（CREATOR）
- 拥有普通用户全部权限
- 创作者等级标识展示在个人主页
- 可能享有更多曝光或推荐

### 管理员（ADMIN）
- 拥有创作者全部权限
- 管理用户账号
- 处理违规内容
- 管理分类体系
- 查看平台统计数据

## 角色标识

- 个人主页会显示 `creator`（是否为创作者）和 `admin`（是否为管理员）标识
- 创作者等级（`creatorLevel`）和等级名称（`creatorLevelName`）也在主页展示

## 角色获取

- 注册后默认为普通用户
- 创作者身份需要满足一定条件后申请
- 管理员由系统指定

## 实名认证

- 接口：`POST /users/me/real-name-verify`
- 实名认证可能影响角色升级和权限获取
