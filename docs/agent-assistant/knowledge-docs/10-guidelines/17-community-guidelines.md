# 社区规范

## 基本原则

CampusShare 致力于打造一个友好、健康、有序的校园资源共享社区。所有用户应遵守以下规范：

## 内容规范

### 允许的内容
- 学习资料、课件、笔记等资源共享
- 学习经验、生活话题讨论
- 有益的知识分享和交流

### 禁止的内容
- 政治敏感内容
- 色情、暴力内容
- 侵犯版权的内容
- 垃圾广告和恶意链接
- 人身攻击和恶意言论
- 虚假信息和谣言

## 行为规范

- 尊重他人，文明交流
- 不刷屏、不灌水
- 不恶意举报
- 不利用平台漏洞谋取利益
- 不冒充他人

## 文件上传规范

- 不上传违规文件
- 不上传恶意软件
- 确保文件不侵犯他人版权
- 文件大小不超过 100MB

## 违规处理

- 违规内容将被删除
- 严重违规者将被封禁账号
- 管理员有权处理违规行为

## 用户角色

- **普通用户（USER）**：基本发帖、评论、互动权限
- **创作者（CREATOR）**：更高级别的内容创作者
- **管理员（ADMIN）**：平台管理权限
